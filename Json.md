### JSON
```js
 Json - текстовый формат обмена данными, используется в Rest API. 
 Json меньше весит и более читабелен для человека.
 ```
 
### Обьект
```js
 Объект - неупорядоченный набор пар ключ/значение.
 Обьект начинается и заканчивается фигурными скобками - {}.
 ```
### Ключ/Значение
```js
 Между "Ключ" и "Значение" ставится ":" - "Ключ":"Значение",
 Пары ключ/значение разделяются  запятой ",".
 Ключ - всегда строка и берётся в кавычки "".
 ```
### Массив
```js
 Массив - упорядоченная коллекция значение. 
 К массиву можно обращаться только номеру элемента, 
 который начинается с нуля "0".
 Массив начинается и заканчивается квадратными скобками "[]",
 ```
### Значения
```js
 Значения разделены запятой ",".
 Значения могут быть: 
 Cтрокой("string", оборачивается в двойные кавычки), 
 Числом(Integer),  
 Логические(True, False),
 Null,
 Объектом или массивом.
 ```

### Пример:
```js
{
	"name": "Isamutdin",

	"city": "Moscow",

	"age": 30,

	"skills": ["Postman",	"SQL", "DevTools", "Git", "Terminal linux", "Charles"	]
};
```
